# 🐢 Python Turtle Learner v1.0

ブロックベースのビジュアルプログラミングでPythonタートルグラフィックスを学習できるWebアプリケーションです。

![Python Turtle Learner](https://img.shields.io/badge/version-1.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## 🌟 主な機能

### 🎨 2つの描画モード

- **フリーハンドモード**: 自由に線を描いて図形を作成
- **マス目モード**: 8x8グリッドでドット絵を作成

### 🎯 豊富なブロック

- **移動**: 前進、後退、円を描く、ホームに戻る
- **回転**: 右回転、左回転
- **ペン**: ペンの上げ下げ、色変更、太さ変更
- **ループ**: 繰り返し処理
- **テンプレート**: 図形（正方形、星、花）、ドット絵（ハート、スマイル、チェッカー）

### 🌈 ファミコン52色パレット

レトロゲーム風の美しい52色から選択可能

### 💾 プログラムの保存・共有

- **保存**: ブラウザに自動保存
- **読込**: 保存したプログラムを読み込み
- **エクスポート**: JSONファイルとして書き出し
- **インポート**: JSONファイルから読み込み

## 🚀 使い方

### 基本操作

1. **ブロックを追加**
   - 左のパレットからブロックをドラッグ＆ドロップ
   - またはブロックをクリックして追加

2. **ブロックを並び替え**
   - プログラムエリアでブロックをドラッグして順序を変更

3. **ブロックを削除**
   - ブロックにマウスを乗せて「✕」ボタンをクリック

4. **プログラムを実行**
   - 「▶ RUN」ボタンをクリック

### モード切り替え

**フリーハンドモード**

- 自由に線を描いて図形を作成
- ペンの太さや色を変更可能
- 円や曲線も描画可能

**マス目モード**

- 8x8のグリッドでドット絵を作成
- セル単位で移動
- `マスを塗る`ブロックでセルを塗りつぶし

### テンプレートの使い方

1. パレット下部の「🌟 テンプレート」セクションを確認
2. テンプレートブロックをプログラムエリアに追加
3. 自動的に複数のブロックに展開されます

## 🎓 学習内容

このアプリケーションを使うことで、以下のプログラミング概念を学べます：

- **順次処理**: ブロックを順番に実行
- **繰り返し**: ループブロックで同じ処理を繰り返す
- **座標系**: タートルの位置と向き
- **関数**: テンプレートブロックで複数の処理をまとめる
- **条件分岐**: ペンの上げ下げで描画を制御
